# Employee Attrition Analysis

## Overview
This project analyzes employee attrition trends using MySQL and Python.

## Tools Used
- MySQL
- Python
- Pandas
- Matplotlib

## KPIs
- Total Employees
- Attrition Rate
- Average Salary
- Average Years at Company

## Visualizations
- Department-wise Attrition
- Job Role Attrition
- Gender-wise Attrition
- Overtime vs Attrition
- Age Group Attrition
- Salary Distribution

## Key Insights
- Identified departments with highest attrition.
- Analyzed impact of overtime on employee turnover.
- Examined attrition across age groups and salary levels.
